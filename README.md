# Home-Cloud

## Requirements
- Python3.x or more: https://www.python.org/downloads/
- flask == 2.1.2
- flask_wtf == 1.1.1
- wtforms == 3.0.1
- werkzeug == 2.1.2
- wheel == 0.38.4

## What is this
A custom-made home cloud storage application build with Python and flask.
